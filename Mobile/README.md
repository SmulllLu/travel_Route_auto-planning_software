# RuleApp

RuleApp，一款功能全面，用户交互良好的文章内容社区程序，兼容安卓苹果端，H5网页，微信小程序等。同时是集成数据本地缓存，邮箱验证，在线投稿（内置Mardown编辑器和富文本编辑器二选一），社会化登录，点赞打赏，积分商城，付费阅读，VIP会员、私聊群聊等一系列功能的自媒体博客资讯APP。

本程序不包含任何按收费项目，功能全部开放且完全开源免费（包括后端接口）。

## 基础功能

1.基于标签，分类，搜索词，文章字段，随机为一体的数据查询调用

2.集成了基于手机短信、邮箱验证的登录注册，邮箱修改，拥有VIP会员体系。

3.集成了在线评论，在线投稿，编辑文章，发布商品等内容操作模块

4.集成了用户签到，打赏，收藏，点赞等基本交互模块，提供邀请返佣等操作，具备完善的收费功能流程。

5.整合积分商城，付费阅读，在线充值（微信、支付宝、易支付、卡密充值、Apple Pay），提现审核。

6.本地列表数据缓存，更优秀的加载体验。

7.兼容安卓苹果APP，H5网页，微信小程序，QQ小程序，并理论兼容所有小程序平台，更多支持正在开发...

8.独有付费广告模块：文章推流广告，横幅广告，启动图广告，轮播广告等（非官方广告联盟）。

9.动态模块，聊天室（私聊，群聊），更多精彩功能正在开发。

10.继承了丰富的安全拦截机制，有效阻止注册机和机器人发布内容。

11.APP内即可后台管理，简约快捷，随手操作。
